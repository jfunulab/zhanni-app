All Areas

Add Address To Account

A user can add address to his account
POST: api/users/{user}/address 201 JSON

Add Card To Account

A user can update his phone number
POST: api/users/{user}/cards 201 JSON

Add Recipient To User Account

A user can save a transfer recipient in his account
POST: api/users/{user}/recipients 201 JSON

Create User

A user can create an account
POST: api/users 201 JSON

Get Bank List

User can get list of available banks
GET: api/banks 200 JSON

Get Bank List Action

Returns list banks available

Get Country List

A user can get list of all countries
GET: api/countries 200 JSON

Get Country State Lists

A user can get list of all countries
GET: api/countries/{country}/states 200 JSON

Request Password Reset

A user can request for email to reset password
POST: api/password/email 200 JSON
Password reset link will not be sent to email that is not registered
POST: api/password/email 422 JSON

Resend User Account Verification Code

A user can request verification code be resent to email
POST: api/email/resend 200 JSON
Unauthenticated user cannot request verification code to be resent to email
POST: api/email/resend 401 JSON
Verified account cannot request for for code to be resent
POST: api/email/resend 422 JSON

Reset Password With Code

A user can reset his password with digit code to email
POST: api/password/reset 200 JSON

Update User

A user can update his phone number
PUT: api/users/{user} 200 JSON

User Login

A user can login with email and password
POST: api/login 200 JSON
A user cannot login with a wrong email
POST: api/login 401 JSON
A user cannot login with a wrong password
POST: api/login 401 JSON
A user is required to provide email to login
POST: api/login 422 JSON
A user is required to provide password to login
POST: api/login 422 JSON

Verify Account

A user can verify their account with code sent to them via email
POST: api/email/verify/{user}/{code} 200 JSON
A user cannot verify their account with a wrong code
POST: api/email/verify/{user}/{code} 422 JSON
A user cannot verify their account with expired code
POST: api/email/verify/{user}/{code} 422 JSON
A user cannot verify account for another user
POST: api/email/verify/{user}/{code} 422 JSON
A user cannot verify their account with code sent to them via email if account is already verified
POST: api/email/verify/{user}/{code} 422 JSON
Unauthenticated user cannot verify account
POST: api/email/verify/{user}/{code} 401 JSON
